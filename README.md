## 📖 Overview

This project is the first assignment in TripleTen's Software Engineering program. The goal was to translate a given design brief into a working webpage using only HTML and CSS — no frameworks, no JavaScript — to build a solid foundation in markup and styling before moving on to more complex projects.

## 🛠️ What I Built & How

I built the Triple Peaks Library page by breaking the design brief down into semantic sections and matching the layout using core CSS techniques rather than relying on a framework. The focus was on writing clean, meaningful markup and controlling layout manually.

**Key features:**
- Semantic HTML structure (proper use of header, nav, section, footer, etc.)
- Custom layout built with CSS positioning and flexible layout systems
- Layering and stacking order managed with `z-index`

**Built with:** HTML, CSS
